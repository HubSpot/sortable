About

Sortable is an open-source javascript and CSS library which adds sorting functionality to tables.

To use, simply add sortable.js and a theme file to your page:

<link rel="stylesheet" href="sortable-theme-default.css" />
<script src="sortable.js"></script>

Add the attribute data-sorting="true" to your <table> elements, and initialize them:

(function(){
    var tables, i, selector;
    selector = 'table[data-sortable="true"]';
    tables = document.querySelectorAll(selector);
    for (i = 0; i < tables.length; i++) {
        sortable.init(tables[i]);
    }
})();
